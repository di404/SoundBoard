# Instant Fun - 网页音效板

这是一个模仿 MyInstants 的极简网页音效板。点击按钮即可播放各种有趣的音效！

## ✨ 功能特点

- **即时播放**：低延迟音频播放，支持打断重播。
- **实时搜索**：快速过滤你想要的音效。
- **响应式设计**：完美适配桌面和移动端。
- **一键停止**：瞬间静音所有声音。

## 🛠️ 技术栈

- 纯 HTML5 / CSS3 / JavaScript
- 无需构建工具，开箱即用

## 🚀 部署到 Zeabur

本项目已针对 Zeabur 进行优化（静态站点）。

1. 将代码推送到 GitHub。
2. 在 Zeabur 中新建项目。
3. 选择 GitHub 仓库进行部署。
4. Zeabur 会自动识别为 HTML 静态网站。
